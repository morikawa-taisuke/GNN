cepsdist.py
目的音声と処理後の音声の誤差を評価する指標
値の範囲 : -∞～∞(正確には無限じゃないです)
(値が高ければ高いほど音声の誤差が小さい)

lesson(1, 2, 3)で引数は3つある
1. 正解とする音声(原音声など)
2. 処理後の音声
3. ファイル名を指定

ファイルの保存先は下記の"../../data_sample/test/"で設定する。
f = open('../../data_sample/test/cd_' + str(files) + '.txt', 'w')


Si-SDR.py
信号同士で比較して分離精度を評価する指標
値の範囲 : -∞～∞(正確には無限じゃないです)
(値が高ければ高いほど音声の分離性能が高い)

calc_sdr(1, 2, 3)で引数は3つある
1. 正解とする音声(原音声など)
2. 処理後の音声
3. ファイル名を指定

ファイルの保存先は下記の"../../data_sample/test/"で設定する。
 f = open('../../data_sample/test/sdr_' + str(files) + '.txt', 'w')


STOI.py
音声明瞭度を評価する指標
値の範囲 : 0～1(値が高ければ高いほど音声が聞き取りやすい)

calc_stoi(1, 2, 3)で引数は3つある
1. 処理後の音声
2. 正解とする音声(原音声など)
3. 保存先とファイル名を指定
